# 🚀 StockShelf Pro - Predictive Portfolio Analytics

![StockShelf Pro Banner](https://via.placeholder.com/1200x400/0f172a/06b6d4?text=StockShelf+Pro+Dashboard)

> **Advanced AI-Powered Portfolio Tracking & Forecasting Dashboard**
> Built with Next.js 14, TypeScript, Tailwind CSS, and Framer Motion.

## 🌟 Key Features

- **🎨 Premium Aesthetic**: Glassmorphism design with sleek gradients and smooth animations.
- **📈 Advanced Visualizations**:
  - **Price Forecasting**: 14-day AI-driven price prediction with confidence intervals.
  - **Risk Analysis**: Efficient Frontier approximation (Risk vs. Return scatter plot).
  - **Technical Indicators**: Real-time RSI tracking.
  - **Asset Allocation**: Dynamic pie chart of your holdings.
- **⚡ Real-Time Interactions**:
  - Seamless "Add/Remove" stock functionality.
  - Interactive charts (zoom, pan, hover).
  - Instant portfolio metric calculations.
- **🛡️ Robust Architecture**:
  - Fully typed with **TypeScript**.
  - Modular component design.
  - **Hybrid Mock Data Engine**: Works perfectly without a backend for demonstration.

## 🛠️ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) + `clsx` + `tailwind-merge`
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Charts**: [React Plotly.js](https://github.com/plotly/react-plotly.js)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/stockshelf-pro.git
    cd stockshelf-pro/frontend
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open your browser**:
    Navigate to [http://localhost:3000](http://localhost:3000) to see the dashboard in action.
